Evaluation workflow

A repeatable process for reliable judgment.

  1. Read the prompt and available context
  2. Map desired output, constraints, and success criteria
  3. Align the rubric with task intent
  4. Inspect model outputs for strengths and failure modes
  5. Compare alternatives and rank what serves the use case best
  6. Apply evaluation criteria consistently across cases
  7. Document evidence, justification, and edge-case notes
  8. Submit a final judgment that can be audited
